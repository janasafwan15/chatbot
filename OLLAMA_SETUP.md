# دليل ربط النظام بـ Ollama

## ما هو Ollama؟

Ollama هو تطبيق يسمح لك بتشغيل نماذج الذكاء الاصطناعي الكبيرة محلياً على جهازك دون الحاجة للاتصال بخدمات سحابية.

## خطوات التثبيت والإعداد

### 1. تثبيت Ollama

#### على Windows:
1. قم بتحميل Ollama من الموقع الرسمي: https://ollama.ai/download
2. قم بتشغيل ملف التثبيت
3. بعد التثبيت، سيعمل Ollama تلقائياً في الخلفية

#### على macOS:
```bash
# استخدام Homebrew
brew install ollama

# أو تحميل التطبيق من الموقع الرسمي
```

#### على Linux:
```bash
curl -fsSL https://ollama.ai/install.sh | sh
```

### 2. تشغيل Ollama

بعد التثبيت، تأكد من تشغيل Ollama:

```bash
# تشغيل Ollama كخدمة في الخلفية
ollama serve
```

سيعمل Ollama على المنفذ الافتراضي: `http://localhost:11434`

### 3. تثبيت نموذج للغة العربية

يُنصح باستخدام أحد النماذج التالية للغة العربية:

#### نموذج Aya (الأفضل للعربية):
```bash
ollama pull aya
```

#### نموذج LLaMA 2:
```bash
ollama pull llama2
```

#### نموذج Mistral:
```bash
ollama pull mistral
```

#### نموذج Gemma 2 (خفيف وسريع):
```bash
ollama pull gemma2:2b
```

### 4. فحص التثبيت

للتأكد من أن Ollama يعمل بشكل صحيح:

```bash
# عرض النماذج المثبتة
ollama list

# اختبار نموذج
ollama run aya "مرحباً، كيف حالك؟"
```

### 5. إعداد النظام

1. افتح التطبيق واضغط على بطاقة "مواطن"
2. في واجهة الشات بوت، اضغط على أيقونة الإعدادات (⚙️) في الأعلى
3. تأكد من أن:
   - عنوان API: `http://localhost:11434`
   - النموذج: `aya` (أو النموذج الذي قمت بتثبيته)
   - درجة الحرارة: `0.7` (للتوازن بين الدقة والإبداع)
4. اضغط على زر التحديث للتأكد من الاتصال
5. إذا ظهرت علامة خضراء "✓ متصل بـ Ollama"، اضغط على "حفظ الإعدادات"

## استخدام النظام

### الوضع الأساسي (بدون Ollama):
- النظام يستخدم قاعدة معرفة محلية ثابتة
- ردود سريعة لكن محدودة

### وضع Ollama (مع AI):
- النظام يستخدم الذكاء الاصطناعي للرد على الاستفسارات
- ردود أكثر ذكاءً وقدرة على فهم السياق
- يمكن الرد على أسئلة متنوعة

## استكشاف الأخطاء

### المشكلة: "غير متصل" في الإعدادات

**الحلول:**
1. تأكد من تشغيل Ollama:
   ```bash
   ollama serve
   ```

2. تحقق من أن المنفذ 11434 غير محظور:
   ```bash
   curl http://localhost:11434/api/tags
   ```

3. أعد تشغيل Ollama

### المشكلة: الردود بطيئة

**الحلول:**
1. استخدم نموذج أخف مثل `gemma2:2b`
2. قلل درجة الحرارة إلى 0.3
3. تأكد من أن جهازك يلبي متطلبات التشغيل

### المشكلة: الردود بالإنجليزية بدلاً من العربية

**الحلول:**
1. استخدم نموذج `aya` المخصص للغات المتعددة
2. تأكد من أن System Prompt يتضمن التعليمة "يرجى الرد باللغة العربية دائماً"

## متطلبات النظام

### الحد الأدنى:
- RAM: 8 GB
- مساحة فارغة: 10 GB
- معالج: أي معالج حديث

### الموصى به:
- RAM: 16 GB أو أكثر
- GPU: بطاقة رسومات NVIDIA (اختياري لكن يسرع الأداء)
- مساحة فارغة: 20 GB
