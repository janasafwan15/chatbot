# 🔌 نظام الدعم الذكي لكهرباء الخليل

نظام دعم عملاء ذكي مدعوم بالذكاء الاصطناعي ومعالجة اللغة الطبيعية (NLP) لخدمة مواطني محافظة الخليل في فلسطين.

## 📋 نظرة عامة

نظام شامل يوفر ثلاث واجهات رئيسية:

### 👥 واجهة المواطنين
- شات بوت ذكي يعمل بالذكاء الاصطناعي
- استفسارات عامة (ساعات العمل، أرقام الاتصال، المواقع)
- إرشادات الاشتراكات والخدمات
- إرشادات الشكاوى والاعتراضات
- الدعم الفني ونصائح توفير الطاقة
- دعم كامل للغة العربية

### 💼 واجهة الموظفين
- نظام تسجيل دخول آمن
- لوحة معلومات بالإحصائيات الحية
- إدارة الردود الذكية (إضافة، تعديل، حذف)
- مراقبة الأداء والتقارير
- تصدير البيانات (Excel, PDF)

### 👨‍💼 واجهة الإدارة
- إدارة المستخدمين والصلاحيات
- تحليلات متقدمة ورسوم بيانية
- مراقبة صحة النظام
- تقارير شاملة قابلة للتصدير

## 🚀 التثبيت والتشغيل

### المتطلبات
- Node.js 18 أو أحدث
- npm أو pnpm

### خطوات التشغيل

1. **استنساخ المشروع**
```bash
git clone [repository-url]
cd hebron-electricity-support
```

2. **تثبيت المكتبات**
```bash
npm install
# أو
pnpm install
```

3. **تشغيل التطبيق**
```bash
npm run dev
```

4. **فتح المتصفح**
افتح المتصفح على: `http://localhost:5173`

## 🔧 ربط النظام بـ Ollama (اختياري)

للحصول على ذكاء اصطناعي متقدم، يمكنك ربط النظام بـ Ollama:

### التثبيت السريع

1. **تثبيت Ollama**
```bash
# Linux/Mac
curl -fsSL https://ollama.ai/install.sh | sh

# Windows: تحميل من https://ollama.ai/download
```

2. **تشغيل Ollama**
```bash
ollama serve
```

3. **تثبيت نموذج عربي**
```bash
# نموذج Aya (الأفضل للعربية)
ollama pull aya

# أو نموذج LLaMA 2
ollama pull llama2
```

4. **إعداد النظام**
- افتح التطبيق واختر "مواطن"
- اضغط على أيقونة الإعدادات ⚙️
- اضبط الإعدادات:
  - عنوان API: `http://localhost:11434`
  - النموذج: `aya`
  - درجة الحرارة: `0.7`
- احفظ الإعدادات

📖 **لمزيد من التفاصيل**: راجع ملف [OLLAMA_SETUP.md](./OLLAMA_SETUP.md)

## 🏗️ البنية التقنية

### التقنيات المستخدمة
- **React 18** - مكتبة واجهة المستخدم
- **TypeScript** - لغة البرمجة
- **Tailwind CSS v4** - تنسيق الواجهة
